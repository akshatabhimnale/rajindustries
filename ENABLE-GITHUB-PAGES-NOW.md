# 🚀 Enable GitHub Pages - Step by Step

Your code is on GitHub, now let's make it live!

## Step 1: Go to Settings

**Click this link:** https://github.com/akshatabhimnale/rajindustries/settings/pages

OR manually:
1. Go to: https://github.com/akshatabhimnale/rajindustries
2. Click the **"Settings"** tab (top menu)
3. Click **"Pages"** in the left sidebar

## Step 2: Configure GitHub Pages

You'll see a page titled "GitHub Pages"

### Under "Build and deployment":

1. **Source:** Click the dropdown and select **"Deploy from a branch"**

2. **Branch:** You'll see two dropdowns:
   - First dropdown: Select **"main"**
   - Second dropdown: Select **"/ (root)"**

3. Click the **"Save"** button

## Step 3: Wait for Deployment

After clicking Save:
- You'll see a message: "GitHub Pages source saved"
- The page will show: "Your site is ready to be published"
- Wait 2-3 minutes

## Step 4: Verify It's Live

Refresh the Settings → Pages page after 2-3 minutes.

You should see:
```
✅ Your site is live at https://akshatabhimnale.github.io/rajindustries/
```

## Step 5: Visit Your Live Website

## Troubleshooting

### "I don't see the Save button"
- Make sure you selected "Deploy from a branch" first
- Then the branch dropdowns will appear

### "It says 404 Not Found"
- Wait a full 5 minutes after enabling
- Clear your browser cache (Ctrl+Shift+Delete)
- Try in incognito mode

### "Build and Deployment is not showing"
- Make sure your repository is Public (not Private)
- Go to Settings → General → scroll to bottom
- Under "Danger Zone", check visibility is Public
